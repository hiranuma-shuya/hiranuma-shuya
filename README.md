## 🚀 My Projects（個人開発）

### [Tablia](https://www.tablia.app/)
**個人飲食店向け予約管理サービス**

高額な既存サービスに代わる、個人飲食店オーナーのための予約管理システム。「飲食店が長く続いてほしい」という想いから、友人のエンジニアと共同運用中。

---

### [酒学道場](https://www.sakegaku-dojo.com/)
**SAKEDIPLOMA試験対策サービス**

年間2,000人が受験する日本酒資格試験の対策サービス。自身の受験経験から、市場に良質な教材がないことを課題に感じ開発。リリース初年度から多くのユーザーに利用され、2026年度は受験者80%へのリーチを目指し、SEO,AEO,広告を頑張っています。

---

### [Pitat-AR](https://www.pitat-ar.com/)
**3DモデルのAR共有サービス**

大手企業のみが導入可能だったAR体験を、誰でも簡単に実現できるサービス。Meta社の3Dモデル自動生成技術との連携を視野に、toC/toB両軸での展開を目指す。
